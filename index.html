Susy

Susy is a Sass powered grid generating framework. In other words it creates scafolding like Twitter Bootstrap, but according to your needs.

Setup

You will need to install ruby, compass and sass and in this instance place them in a gulp pipeline.

Below is a basic setup example. For more information visit the projects homepage: susy.oddbord.net 

$susy: (
  columns: 12,
  gutters: 1/4,
  math: fluid,
  output: float,
  gutter-position: inside,
);

Mixin - Isolate

Isolate can create a very cool effect, when combined with Breakpoints it lets you at some point in your layout have a more asymetricaly layout, that does not affect the responsivness of your site negativley. This div utilizes Isolate() at medium browser widths
